날것(Raw)의 철학

성공 사례만 나열하는 강의가 아닙니다. 실패한 광고비, 망한 상품 기획, 반품 폭탄의 원인까지 — 가감 없이 꺼내놓는 현장 언어로 진행합니다.

⚡

실전 중심 커리큘럼

이론서에 없는 매출 공식. 오늘 들은 내용을 내일 바로 써먹을 수 있는 액션 플랜 중심으로 설계된 밀도 높은 프로그램입니다.

🎯

타겟 명확화

막연하게 "쇼핑몰 해볼까" 하는 분부터 정체 구간에서 다음 단계를 찾는 운영자까지 — 각자의 단계에 맞는 돌파구를 제시합니다.

🤝

소수 정예 네트워킹

50인 이내 소규모 운영으로 질문 제한 없는 Q&A, 참가자 간 수평적 네트워킹이 가능한 밀착 환경을 보장합니다.

# 날것의 현장성 # 0→매출 로드맵 # 실패를 말하는 용기 # 이커머스 # 쇼핑몰 창업 # 성장 전략

01

오프닝 — 시작의 기술, 더잠·포들 그리고 롤리까지

세 개의 브랜드 론칭 경험으로 말하는, 신생 브랜드가 가장 먼저 할 것 vs 하지 말아야 할 것. 무료배송·가격 전에 선행되어야 할 브랜딩 작업, 그리고 실제 타임라인 공개 (런칭 D-day ~ 첫 매출).

# 브랜드 론칭 # 브랜딩 선행 # 실전 타임라인 # 할 것 vs 하지 말 것
오프닝
02

메인 — 광고 없이 첫 매출 만들기

무자본으로 인플루언서를 섭외하는 실전 DM 스크립트 공개. 팔로워 수보다 중요한 것 — SNS · 커뮤니티 · 바이럴을 활용해 돈 없이 첫 매출을 만드는 법.

# 무자본 인플루언서 # DM 스크립트 # SNS/커뮤니티 # 바이럴
메인 주제
03

심화 — 매출 10배 스케일업, 본격적인 매출에 필요한 것들

자사몰 운영 실무 (CS · 재고 · 광고 세팅 노하우) 총정리. 실제 사용 중인 AI 툴 소개 — 상세페이지 · 챗봇 · 광고 소재 제작까지.

# 자사몰 운영 # CS/재고/광고 # AI 툴 활용 # 스케일업
심화 내용
